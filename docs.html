Your complete guide to managing ARK-based wallets, securing your assets, and navigating the ARK blockchain.

# Introduction

ARK Desktop Wallet is a powerful, user-friendly application for managing ARK-based cryptoassets. Built for Windows and macOS, it combines secure offline key storage with seamless integration into the ARK ecosystem.

Key Features:
  • Create and manage ARK-based wallets.
  • Send and receive ARK transactions securely.
  • Vote for delegates and participate in governance.
  • View and manage multiple cryptoassets in a single interface.

# Overview

The ARK Desktop Wallet provides:
  • 24-Word Recovery Phrases: Each wallet is secured with a unique BIP39 mnemonic for recovery.
  • Password Encryption: Locally encrypt your wallet credentials for enhanced security.
  • Offline Storage: Private keys remain offline, ensuring your funds are always safe.
Warning

Losing your 24-word recovery phrase means permanent loss of access to your wallet. Keep it secure and never share it.

# Create a Wallet

Steps to Create a Wallet:
  1. Open the ARK Desktop Wallet and select “Create Wallet”.
  2. Generate a Wallet Address: A unique address will be created.
  3. Secure Your Recovery Phrase: Write down the 24-word mnemonic and verify it.
  4. Set a Password: Create an encryption password to protect local access.
  5. Complete the setup and start managing your ARK assets.

# Import a Wallet

Methods to Import Wallets:
  • 24-Word Recovery Phrase (recommended).
  • BIP-38 Encrypted Wallet.
  • Public Key or Address.
Steps to Import:
  1. Select “Import Wallet” from the main menu.
  2. Enter the required credentials.
  3. Confirm and set an encryption password to secure access.
Information

Always import wallets using trusted devices and ensure your recovery phrase is private.

# ARK Transaction Features

With ARK Wallet, you can:
  • Send Funds: Enter the recipient’s wallet address, set the transaction amount, and confirm securely.
  • Receive Funds: Generate a QR code or copy your address to share with senders.
  • Track History: View detailed transaction information, including confirmations and fees.

# ARK Transaction Types

Supported transaction types include:
  • Standard Transfers: Send ARK tokens between wallets.
  • Delegate Voting: Vote for network validators to secure the ARK blockchain.
  • Multi-Signature Transactions: Add co-signers for added security.
Advanced Options:
  • VendorField Messages: Attach messages to your transactions for additional context.

# Security

Supported transaction types include:
  • Offline Private Key Storage: Private keys never leave your device.
  • Local Password Encryption: All credentials are encrypted locally.
  • Secure Backups: Use your 24-word recovery phrase to restore wallets.
Pro Tip:

Regularly update your wallet to access the latest security features and improvements.
