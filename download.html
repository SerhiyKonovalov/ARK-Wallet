Update History

Release 1.06.03
New features:
  1. Important security updates
  2. New assets: TIA, AXL, DYM, AKT, NIBI
  3. Loading speed significantly improved
Bug fixes:
  1. TON sending from an inactive address error
  2. BTC incorrect network fee calculation
  3. ATOM stake/claim operations unavailable issue
  4. Issues with ETHOP, ETHARB exchanges resulting in ETH exchange
  5. Lag while searching for asset
Release 1.05.04
New features:
  1. Restake option for INJ and LUNC staking has been added. No longer waiting until the unbonding period is over to change the validator.
  2. Password strength indicator has been implemented during wallet creation and changing the current password.
Bug fixes:
  1. Minor visual bugs fixed
  2. Set fee button resulting in FTM gas price fluctuation fixed
Release 1.04.08
New features:
  1. LUNA staking is now supported. Stake your coins with up to 7% APY.
  2. NEO2 and GAS2 were delisted.
  3. XMR View Key button is added to the XMR page. You can now easily contact support regarding your XMR issues
Bug fixes:
  1. Staking balances displaying bug is fixed.
  2. Monero XMR balance, history, sending and exchanging fixed
  3. Flare staking rewards balance fixed
  4. USDT-TRX sending fixed.
Release 1.02.10
New features:
  1. Optimism L2 network has been integrated to Atomic Wallet.
  2. Manage & Exchange Optimism OP token in a secure way.
Bug fixes:
  1. Internet connection established alert is reflected again.
Release 1.01.03
New features:
  1. Arbitrum L2 network has been integrated to Atomic Wallet
  2. Manage & Exchange Arbitrum ARB token in a secure way.
  3. FlareDrops is now supported. Activate the drop in the wallet and enjoy monthly FLR drops.
Bug fixes:
  1. Minor performance improvements
  2. Staking page showing a blank screen is fixed.
  3. TRX-USDT sending has been fixed.
